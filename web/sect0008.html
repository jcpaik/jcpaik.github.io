2.2 Monotone Sofas

We observed in Theorem 2.6 that any sofa \(S\) with rotation angle in \([0, \pi / 2]\) can be repositioned to have a monotone movement \(f_t\). Equation 1.1 then tells us that \(S\) is contained in the following intersection of rotated stripes and hallways.

\begin{equation} \label{eq:monotone-sofa} M(S) = H \cap R_\beta (V) \cap \bigcap _{0 \leq t \leq \beta } L_t \end{equation}
2.1

We show that this set is itself a sofa.

Theorem 2.7

Let \((S, f_t)\) be a sofa with monotone movement, so that it has a rotation angle \(\beta \in [0, \pi / 2]\) and sofa (and hallway) movement of \(f_t\) (and \(g_t = f_t^{-1}\)) respectively. Denote the moving sofa as \(S_t = f_t(S)\) and the moving hallway as \(L_t = g_t(L)\). Then the set \(M(S)\) containing \(S\) is itself a sofa with movement \(f_t\).

Proof ▼

The set \(M(S)\) is closed by definition, and it is bounded as \(H \cap R_\beta (V)\) is bounded for \(\beta {\lt} 0\) and \(H \cap L_{\pi / 4}\) is bounded for \(\beta = \pi / 2\). So it suffices to show that \(M(S)\) is connected. Then, by Definition 1.13, the movement \(f_t\) is a valid movement of \(M(S)\) and so \((M(S), f_t)\) is a sofa.

Fix an arbitrary point \(v \in N(S)\). We will show that there is a line segment inside \(M(S)\) connecting \(v\) to a point in \(S\). This completes the proof of connectedness of \(M(S)\), as \(M(S)\) is the union of \(S\) with line segments extending from \(S\).

Let \(l_\theta \) be the line through \(v\) that makes an angle \(\theta \in [\beta , \pi / 2]\) in counterclockwise direction from the \(x\)-axis. Regardless of \(\theta \), the intersection \(s_\theta = l_\theta \cap M(S)\) is either a line segment or an empty set. This is because \(s_\theta \) is the intersection of \(l_\theta \cap H\), \(l_\theta \cap R_\beta (V)\), and \(l_\theta \cap L_t\) for all \(t \in [0, \beta ]\), all line segments contained in \(l_\theta \).

Now it only remains to show that for some \(\theta \), \(l_\theta \) overlaps with \(S\), so that the segment \(s_\theta = l_\theta \cap M(S)\) connects \(v\) with a point in \(S\). If either one of \(l_\beta \) or \(l_{\pi / 2}\) overlaps with \(S\), then we are done. Assume otherwise that none of \(l_\beta \) and \(l_{\pi / 2}\) overlaps with \(S\). Since \(S\) is nonempty and connected, it is either on the left side of \(l_{\pi / 2}\) or the right side of \(l_{\pi / 2}\). As we are working with the monotone movement of \(S\), \(S\) touches the side \(a(0)\) of \(L_0\) inside \(L_0\). This implies that \(S\) is on the right side of \(l_{\pi / 2}\). A similar argument with the side \(c(\beta )\) of \(L_\beta \) shows that \(S\) is on the left side of \(l_\beta \). Now move \(l_\theta \) continuously, taking \(\theta \) from \(\beta \) to \(\pi / 2\). As \(S\) is on the different sides of \(l_\theta \) for \(\theta = \beta \) and \(\pi / 2\), at some point the line \(l_\theta \) will pass through \(S\). This completes the proof.

Definition 2.8

Let \((S, f_t)\) be any sofa that follows the setting of Theorem 2.7. Also, let \(P = H \cap R_\beta (V)\) be the parallelogram (or strip for \(\beta = \pi / 2\)). Rewriting \(L_t\) as \(Q^+_t \setminus Q^-_t\) where \(Q^+_t = g_t(Q^+)\) and \(Q^-_t = g_t(Q^-)\), we have the following equality.

\begin{align} \label{eq:monotone-sofa-structure} M(S) & = H \cap R_\beta (V) \cap \bigcap _{0 \leq t \leq \beta } L_t \\ & = \left(P \cap \bigcap _{0 \leq t \leq \beta } Q^+_t \right) \setminus \bigcup _{0 \leq t \leq \beta } \left(P \cap Q^-_t \right) \end{align}

Definition 2.9

Let \((S, f_t)\) be any sofa with monotone movement and rotation angle in \([0, \pi / 2]\). Define the cap of \(S\) as

\[ \mathcal{C}(S) = P \cap \bigcap _{0 \leq t \leq \beta } Q^+_t \]

and the niche of \(S\) as

\[ \mathcal{N}(S) = \bigcup _{0 \leq t \leq \beta } \left(P \cap Q^-_t \right). \]

For each \(t \in [0, \beta ]\), let \(T_t = P \cap Q^-_t\) be the wedge of \(S\) at angle \(t\). Consequently, \(M(S) = \mathcal{C}(S) \setminus \mathcal{N}(S)\) and \(\mathcal{N}(S) = \cup _{0 \leq t \leq \beta } T_t\).

Definition 2.11

Any sofa \(S\) with monotone movement \(f_t\) is a monotone sofa if \(S\) is exactly same as the set \(M(S)\) in Equation 2.1.

Theorem 2.12

Any sofa of rotation angle in \([0, \pi / 2]\) is, after some reposition by some orientation-preserving isometry, is contained in a monotone sofa.

Proof ▼

Take any sofa \(S\). After some reposition, \(S\) admits a monotone movement \(f_t\) by Theorem 2.6. By Theorem 2.7, the shape \(M(S)\) is also a sofa with the same movement \(f_t\). By Theorem 2.10, the support function of \(S\) and \(M(S)\) matches. Now note that a monotone movement solely depends on the support function of the sofa’s shape by definition. So \(f_t\) is not only the monotone movement of \(S\), but also of \(M(S)\). Now \((M(S), f_t)\) is a sofa with monotone movement. Consequently, \(M(M(S)) = M(S)\) and so \((M(S), f_t)\) is a monotone sofa.

Thus, it suffices to look for monotone sofas if we want to find a sofa of maximum area.

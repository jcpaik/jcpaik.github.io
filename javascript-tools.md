- `node`: Runtime environment for javascript - that is, makes javascript executible. 
    Has a LOT of versions and very likely incompatible with each other.
- `nvm`: Manages different versions of `node`. Can download different versions, choose which one to use, etc.
- `npm`: Default package manager for `node`. One for each version of `node`
- `yarn`: Another package manager for `node` that also serves as project manager - can install packages project-wise.
